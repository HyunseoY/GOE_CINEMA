# 📽️GOE’s CINEMA

개인과제 영화검색 사이트 만들기

### 1. 소개

<p>영화 검색 웹사이트.</p>
<p>인기순, 현재상영작, 높은평점순 등으로 정렬된 영화들을 볼 수 있고,</p> 
<p>특정 영화를 검색할 수 있다.</p>
<p>디자인은 ‘레트로’ 와 ‘시네마’ 키워드를 착안. 붉은색과 전구로 장식된 이미지를 바탕으로 한다.</p>
